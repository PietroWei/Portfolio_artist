# AI Artist Portfolio (GitHub Pages)

Sito statico multi-pagina pronto per GitHub Pages.

## Pagine
- `index.html` (Home)
- `profilo.html`
- `progetti.html`
- `contatti.html`

## Deploy su GitHub Pages
1. Crea un repository su GitHub (es. `ai-artist-site`).
2. Carica tutti questi file nella root del repository.
3. Vai in **Settings > Pages**.
4. In **Build and deployment**, scegli:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (root)
5. Salva e attendi la pubblicazione.

Il sito usa risorse esterne (immagini/video embed), quindi non richiede build o dipendenze.
